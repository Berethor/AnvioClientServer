Установка службы:
1)Запустить командную строку с правами администратора(Пуск+R -> ввести cmd -> ctrl+shift+enter)
2)Перейти в директорию с Install_Service.bat через cd через "cd Путь до файла"
3)Запустить Install_Service.bat

Удаление службы:
1)Запустить командную строку с правами администратора(Пуск+R -> ввести cmd -> ctrl+shift+enter)
2)Перейти в директорию с Delete_Service.bat через "cd Путь до файла"
3)Запустить Delete_Service.bat
4)Перезапустить компьютер

Возможные проблемы:
В случае если установлена 32 битная система изменить в Install_Service.bat\Delete_Service.bat Framework64 на Framework
